# Диспенсер кошачьего корма

## Описание проблемы
Диспенсер кошачьего корма может отличать особь по окрасу и морде, но не может запретить выдачу еды, поэтому будет 
присылать сообщение раз в день с отчетом о том какой кот и сколько еды поглотил за сутки. 
В свою очередь Система будет считывать сообщения и передавать в консоль если какой-то из котов перевыполнил 
или недовыполнил норму по еде в сутки.

### Объекты наблюдения

#### Кошка Кира, недовес
Ест мало, шугается Феликса, может недоедать из-за него свою норму.

#### Кот Феликс, еще чуть чуть и будет ожирение
Чаще всего отнимает еду у Киры пока хозяева не смотрят. Врачи советуют ему поменьше жрать, но его это совершенно не волнует.
